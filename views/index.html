Timestamp Microservice

A full stack JavaScript app that returns Unix and UTC timestamps for a given date.

Usage

  • GET /api/ - Returns the current time
  • GET /api/2015-12-25 - Returns timestamp for the date
  • GET /api/1451001600000 - Returns timestamp for the Unix time (ms)

Example Output

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Error Example

{
  "error": "Invalid Date"
}
